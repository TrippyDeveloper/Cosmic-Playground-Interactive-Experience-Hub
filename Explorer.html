Fractal Explorer Controls

  • Mouse Controls:
    • Click and drag to pan around the fractal
    • Scroll wheel to zoom in and out
    • Double-click to zoom in at a specific point
  • Keyboard Shortcuts:
    • + or ] - Zoom in
    • - or [ - Zoom out
    • Arrow Keys - Pan around
    • R - Reset view
    • S - Save image
    • H - Toggle this help
    • 1-4 - Switch between fractal types
  • Fractal Types:
    • Mandelbrot Set: The classic fractal based on the equation z = z² + c
    • Julia Set: Related to the Mandelbrot set but with a fixed complex parameter
    • Burning Ship: A variation that uses the absolute values in the calculation
    • Tricorn: Also known as the Mandelbar, uses complex conjugation
  • Tips:
    • Higher iteration values give more detail but render slower
    • Try different color schemes to reveal different patterns
    • Explore the preset locations to find interesting patterns
    • The Julia set changes dramatically with different parameters
